# Foundation

This page documents what Milestone 1 established. It is historical context, not the current delivery state.

## What M1 Added

- Bun monorepo structure
- Local harness state and progress loop
- Initial PRD, architecture, ADR, and GitBook layout
- CI and shared repo tooling

## What Came Later

The repository now includes:

- Hono API routes for strategies, purchases, recommendations, bundles, credits, identity, analytics, leaderboard, execution, and health
- Astro pages for home, strategies, bundles, leaderboard, status, and OpenClaw docs
- A dedicated backtest worker
- Shared contracts and D1 schema packages

For the live implementation snapshot, use [Implementation Status](implementation-status.md) and `docs/PROGRESS.md`.
